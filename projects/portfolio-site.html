Portfolio Website

Mijn persoonlijke IT-portfolio met projecten en vaardigheden

Over dit project

Deze portfolio-website is volledig zelf gebouwd in HTML en CSS. De site toont wie ik ben, welke IT-vaardigheden ik bezit, en welke projecten ik gerealiseerd heb tijdens mijn opleiding tot systeembeheerder.

Doel van dit project

Een professionele online aanwezigheid creëren waarmee ik mezelf kan voorstellen aan IT-recruiters en werkgevers. Ik wilde een centraal punt waar alles samenkomt: projecten, skills, CV en contactinfo.

Gebruikte technologieën

Wat ik geleerd heb

  • Semantische HTML en gestructureerde CSS
  • Azure Static Web Apps deployment
  • CI/CD pipelines via Azure DevOps
  • Branches gebruiken (test → main workflow)
  • DNS koppeling via Azure DNS Zones + domein van Combell
