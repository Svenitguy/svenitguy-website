Mijn IT Lab

Een volledig zelfgebouwde IT-omgeving met networking, servers, virtualisatie en cloud automatisatie. Dit lab simuleert een echte enterprise infrastructuur.

Lab infrastructuur

  • 1x TP-Link TL-ER6120 Router
  • 1x TP-Link T2600G-28TS (Layer 3 Core Switch)
  • L2 Switches: T1500G-10MPS, Cisco SG300-20, TL-SG1016DE, Cisco SG350-10MP
  • WiFi: 2x TP-Link EAP225 + Omada OC200
  • IP Camera’s: Reolink RLC-423 + RLC-420
  • NAS: QNAP TS-431XeU (CCTV), QNAP TS-832XU (lab), QNAP TS-469L (backup)
  • Servers: Supermicro 5018A-TN4 (Windows Server + Hyper-V), Custom i7-2600K ESXi host
  • Workstations: 4 desktops + 3 laptops

Netwerkdiagram

Mijn netwerk is opgebouwd als een gesegmenteerde omgeving met VLANs, routing en switching binnen een 18U rack.

  • VLAN-segmentatie en netwerkarchitectuur
  • Inter-VLAN routing via L3 core switch
  • Netwerksegmentatie (CCTV vs lab vs home)

  • Virtualisatie met VMware ESXi en Hyper-V
  • Nested labomgevingen op workstation
  • Testomgevingen voor Windows Server en automatisatie

Cloud & Automatisatie

Vaardigheden ontwikkeld in mijn labomgeving

Mijn homelab werd gedurende vele jaren gebruikt om praktijkervaring op te bouwen met netwerken, systeembeheer, virtualisatie, storage en infrastructuur. Deze praktijk heb ik verdergezet via virtuele labomgevingen op mijn desktop en cloudprojecten in Microsoft Azure.

  • Ontwerp en beheer van een gesegmenteerd netwerk met VLAN's, routing en switching
  • Netwerksegmentatie voor thuisgebruik, labomgevingen en CCTV-infrastructuur
  • Opzetten van Windows Server omgevingen met Active Directory, DNS, DHCP en Group Policies
  • Gebruikersbeheer, rechtenstructuren en gedeelde resources binnen domeinomgevingen
  • Uitbouwen van multi-server labs met domeincontrollers, fileservers en Windows clients
  • Virtualisatie met VMware ESXi en Microsoft Hyper-V voor test- en leeromgevingen
  • Beheer van NAS-opslag voor data, camerabeelden, labo's en back-updoeleinden
  • Toepassen van redundantie- en back-upstrategieën binnen een thuisinfrastructuur
  • Verderzetten van virtualisatie- en infrastructuurlabs op een desktopomgeving met 64 GB RAM
  • Uitbreiding naar cloudtechnologieën met Microsoft Azure, Terraform en GitHub CI/CD
